# Case Study: Teddies Restaurant Website

## Problem
Small restaurants often lack:
- modern online presence
- direct reservation systems
- mobile-optimized menus

---

## Solution
Built a fully responsive restaurant website with:

- interactive menu filtering
- gallery showcase system
- reservation form (Netlify-powered)
- conversion-focused UX structure

---

## Design Approach

Focused on:
- warmth (woodland theme)
- trust (family storytelling)
- simplicity (fast navigation)
- conversion (clear booking flow)

---

## Key Features

### Menu System
Users can filter meals by category instantly.

### Reservation Flow
Form → submission → success page → confirmation number.

### Gallery
Visual-first browsing experience with lightbox interaction.

---

## Outcome

A reusable restaurant website template suitable for:
- cafés
- family restaurants
- small hospitality businesses

---

## Future Improvements

- online payment integration
- admin dashboard
- CMS integration (Sanity/Contentful)
